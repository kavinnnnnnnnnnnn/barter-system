# 🔐 Exchange Documentation & Verification Guide

## Overview

The Barter System Exchange now requires **secure verification and documentation** to ensure trust and compliance. All users must provide identity verification before completing an exchange.

---

## 📋 Required Documentation

### 1. **Phone Number** 📱
- **Purpose**: Contact verification and account recovery
- **Format**: Include country code (e.g., +91 9876543210)
- **Validation**: Minimum 10 digits with numbers and country code
- **Examples**:
  - ✅ +91 9876543210
  - ✅ +1 2025551234
  - ✅ 9876543210 (India)
  - ❌ 12345 (too short)

---

### 2. **Aadhar Card Proof** 🆔
- **What is it?**: Unique 12-digit identification number issued by UIDAI (India)
- **Who needs it?**: Indian residents
- **Accepted formats**: JPG, PNG, PDF
- **File size limit**: Maximum 5MB
- **What to upload**: Clear, legible copy of:
  - Front side of Aadhar card
  - Scanned copy or photograph
  - Both sides if possible

**Upload Guidelines**:
- Ensure document is clearly visible
- All text must be readable
- No blurry or partial images
- Avoid highlighting or marking

---

### 3. **PAN Card Proof** 🏦
- **What is it?**: Permanent Account Number for taxation (India)
- **Who needs it?**: All Indian taxpayers
- **Accepted formats**: JPG, PNG, PDF
- **File size limit**: Maximum 5MB
- **What to upload**: Clear copy of:
  - Front side of PAN card
  - Scanned copy or photograph
  - Both sides if possible

**Upload Guidelines**:
- PAN number must be clearly visible
- Ensure document is not expired
- High quality, legible image
- All corners should be visible

---

### 4. **International Proof** (Optional) 🌍

**When Required**:
- International users
- Cross-border exchanges
- Users with foreign transactions

**Accepted Document Types**:

#### A. **Passport**
- Valid passport copy
- Photo page must be clear
- Passport number visible
- Expiry date visible
- Accepted formats: JPG, PNG, PDF

#### B. **International Driving License**
- Valid driving license from home country
- License number visible
- Expiry date visible
- Photo must be clear
- Accepted formats: JPG, PNG, PDF

#### C. **Visa Copy**
- Visa stamp or document copy
- Destination country visible
- Validity dates clear
- Your name visible
- Accepted formats: JPG, PNG, PDF

#### D. **Other International Document**
- Work permit
- Residence permit
- Student visa
- Travel document
- Any official government-issued document

## ✅ Upload Process

### Step 1: Select Document Type (for International Proof)
```
1. Click "International Identity Proof (Optional)"
2. Select document type from dropdown:
   - Passport
   - International Driving License
   - Visa Copy
   - Other International Document
3. Upload field appears below
```

### Step 2: Upload File
```
1. Click "Choose File" or drag-and-drop
2. Select document from your device
3. Ensure file is under 5MB
4. File preview shows in field
```

### Step 3: Verify Information
```
1. Review all entered information
2. Check "I verify that all documents..."
3. Click "Confirm Exchange Request"
4. Wait for upload confirmation
```

## 📸 File Upload Requirements

### Supported Formats
- ✅ **JPG** (.jpg, .jpeg)
- ✅ **PNG** (.png)
- ✅ **PDF** (.pdf)

### File Size Limits
- **Aadhar Card**: Maximum 5MB
- **PAN Card**: Maximum 5MB
- **International Proof**: Maximum 5MB
- **Total**: Up to 15MB for all documents

### Image Quality Standards
- **Minimum Resolution**: 800×600 pixels
- **Recommended Resolution**: 1200×800 pixels or higher
- **Format**: Color or black & white both acceptable
- **Contrast**: High contrast for better readability

### Document Visibility
- ✅ All text clearly readable
- ✅ No glare or reflections
- ✅ Complete document visible (all corners)
- ✅ Proper lighting
- ✅ Not cropped or cut off

---

## 🚫 What NOT to Upload

❌ **Expired Documents**
- Verify document expiry date before upload
- Expired documents will be rejected

❌ **Blurry or Low Quality**
- Ensure high contrast and clarity
- Use good lighting when photographing

❌ **Partial Documents**
- Upload complete document with all corners visible
- Don't crop important information

❌ **Edited or Modified Documents**
- Original documents only
- No Photoshop or digital modifications

❌ **Someone Else's Documents**
- Only your own documents accepted
- Documents must match account name

❌ **Screenshots or Photos of Screens**
- Use direct scans or photographs
- Not screenshots from emails

## 🛠️ Troubleshooting

### Issue: File Too Large
**Solution**:
1. Compress image using online tools
2. Reduce image resolution
3. Convert JPG to PDF
4. Maximum 5MB per file

### Issue: File Won't Upload
**Solution**:
1. Check file format (JPG, PNG, PDF)
2. Verify file size under 5MB
3. Try different browser
4. Clear browser cache
5. Try uploading again

### Issue: Image Too Blurry
**Solution**:
1. Retake photograph with better lighting
2. Use scanner instead of camera
3. Increase image resolution
4. Ensure document is flat and straight

### Issue: Document Rejected
**Solution**:
1. Check why rejected (email notification)
2. Resubmit with better quality
3. Ensure all text is readable
4. Verify document is not expired
5. Check name matches account

### Issue: Verification Pending Too Long
**Solution**:
1. Documents usually verified in 1-2 hours
2. Check email for rejection reason
3. Contact support if delayed beyond 24 hours
4. Provide order ID/reference number
